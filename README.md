# 🏆 EduGamify - Frontend

Sistema de Gamificação Educacional para aumentar o engajamento de alunos através de pontos e recompensas.

## 📋 Sobre o Projeto

O EduGamify é uma plataforma que transforma atividades educacionais em desafios recompensadores, permitindo que professores gerenciem pontos e recompensas para motivar os alunos.

### Funcionalidades

#### 👨‍🏫 Professor
- Configurar pontuação para presença, tarefas e participação
- Registrar atividades rapidamente
- Gerenciar alunos (adicionar, visualizar pontos)
- Aprovar/rejeitar solicitações de resgate
- Gerenciar catálogo de recompensas

#### 👨‍🎓 Aluno
- Visualizar saldo de pontos
- Ver ranking da turma
- Acompanhar histórico de atividades
- Solicitar resgate de recompensas
